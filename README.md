# OTOWA IoT Operation Monitor

## 概要
製品開発資産を一元管理するための標準リポジトリです。  
回路図、ファームウェア、PCソフト、試験結果、リリース資産を統合管理します。

## フォルダ構成
- `project/` : 仕様書・計画書
- `circuit/` : 回路図・部品表
- `simulation/` : 解析・シミュレーション
- `firmware/` : 組込みソフト
- `software/` : PCツール・Javaアプリ
- `test/` : 試験結果・画像
- `release/` : リリース版成果物

## プロジェクト目的
開発資産の見える化、標準化、技術継承、品質トレーサビリティを実現する。

## スローガン
**変化を楽しみ、未来を創ろう**
